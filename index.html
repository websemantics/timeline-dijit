Timeline-dijit

Dijit Widget for audio/video timeline navigation

  • Download ZIP
  • Download TAR

This project is maintained by Websemantics, Inc.

Timeline-dijit is a Dijit Widget for timeline representation of <audio /> or <video /> HTML5 tag; based on timeline-js Read the examples with Chrome or Firefox. This is not a library limitation, but only a codec limitation : examples use OGG files from wikimedia.
